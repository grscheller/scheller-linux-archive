In Python:
  Convert Hex String to decimal Int:
    >>> int("03bb", 16)
    955
  Convert an Int to a hex string:
    >>> hex(955)
    '0x3bb'

Bash, vim and most modern linux terminal windows are unicode aware:
    $ <ctrl>+<shift>+u03bb<ret>
    $ λ
but on the same line.

Example super and subscripts:
<U-03bb><U-2080><U-207f> -> λ₀ⁿ

(pointless for fixed sized fonts)
