Reimaging a Mac mini:

0. Make sure Apple keyboard are plugged into computer and turned on.
1. Make sure you have Ethernet cable plugged with connectivity to Internet.
2. To reimage a Mac mini, power off system, hold down key
   combos while rebooting.

   - Opt-Cmd-R to install latest compatible MacOS
   - Opt-Shift-Cmd-R to install original OS that came with computer

2. Once booted into "recovery app", unplug mouse & keyboard.
3. In the recovery app window, select Reinstall macOS.
4. Fuble through idiot online instructions.
